Vaccine Reservation

The Vaccine Reservation System is an online platform for vaccine appointments, comprising backend management (Java, Spring Boot) and frontend user interfaces (JavaScript, React) using a frontend-backend separation approach. Check out my Project.

Key Technical Features:

  • Built on SpringBoot and SpringCloud for a sturdy microservices framework.
  • Uses NoSQL, with Redis for fast data caching and MongoDB for high-concurrency operations.
  • Incorporates RabbitMQ for enhanced order concurrency.
  • Scheduled reminders ensure timely medical alerts.

Integrations:

  • Alibaba Cloud OSS for data storage.
  • SMS for notifications.
  • WeChat for login, payments, and refunds.
A streamlined system tailored for efficient vaccine reservations.

PACCAR

Our research centers on enhancing object detection in autonomous vehicles. Traditional methods and neural networks struggle with detecting entities like pedestrians and bicycles. We refine a deep learning neural network based on the Yolov5 algorithm by adjusting hyperparameters and layers. Our innovation lies in incorporating thermal cameras known for detecting objects in darkness, fog, and adverse conditions. Merging thermal data with Lidar, radar, and visible light inputs creates a comprehensive system for object classification through machine learning. Deployed on NVIDIA Xavier GPUs, our final model achieves a remarkable 142-fps real-time processing rate, swiftly categorizing objects. Notably, our model achieves 93.7% accuracy in identifying various road users. This approach enhances autonomous vehicles' detection capabilities, bolstering their safety and efficiency.
